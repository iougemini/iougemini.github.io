安卓手机自建 KMS 服务器（无需 Root）


1. 下载并安装 v0v.bid 提供的安卓版 KMS 服务器软件 点击下载

2. 确保手机与需要激活的电脑连接同一 WIFI 网络 (或在同一局域网内)，启动软件

3. 点击 [启动服务器]，查看软件内显示的 [IP 地址]

4. 使用如下代码即可激活你手机所在局域网内的所有 Windows 系统和 office 套件 (注意替换为自己的 IP 地址)

命令提示符(管理员)：slmgr /skms 192.168.1.XXX && slmgr /ato

Linux / Windows 系统自建KMS服务器请访问：

https://github.com/dylanbai8/kmspro
