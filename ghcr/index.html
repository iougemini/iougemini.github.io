GHCR.IO 镜像加速

仅CN IP可以使用

使用 docker hub proxy

将原拉取镜像命令：

docker pull ghcr.io/helloworld/helloworld

替换为下面的加速拉取镜像命令：

docker pull ghcr.anyhub.us.kg/helloworld/helloworld:1.0.3

本站使用Cloudflare CDN进行加速，想要获得更好的速度，可以使用优选IP。
